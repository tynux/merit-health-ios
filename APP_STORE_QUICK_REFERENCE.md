## ⚠️ 常见错误

| 错误 | 原因 | 解决方案 |
|------|------|---------|
| 签名无效 | 证书或预配置文件过期 | 重新创建证书和预配置文件 |
| 构建失败 | Bundle ID 不匹配 | 验证 Bundle ID 配置 |
| 审核被拒 | 不符合审核指南 | 阅读拒绝原因并修复 |
| 上传失败 | 网络问题 | 检查网络连接并重试 |
